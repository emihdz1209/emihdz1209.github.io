2.CSS

  • Explain a:
    • Tag Selector

      • A selector is an attribute that can help us modify the stylistic values of more than one element at a time.

    • Class Selector

      • A class is a way to modify the style of a group of elements at once.

    • Tag and Class Selector

      • You can use both selectors to modify the style of a group of elements at once.

    • ID Selector

      • An ID is a unique identifier for an element, and can be used to modify the style of a single element.

  • What is a class? What is an id? How are they related to CSS?

    • Classes and IDs work along selectors to help modify the style of various elements at once. This helps write cleaner Hypertext. An ID will be unique to each element, while classes are better suited to modify a large number of them at once.

  • Explain the CSS Box Model? Show an example.

    • The CSS box model is the way in which containers work in HTML. You can use either grids of flex containers to hold greater control over the display of content, and using specific modifiers, you can even change the direction and space between and around all the items inside your boxes.

  • How can you change the font color using CSS?

    • Usualy, modify the "color" of any container that holds any text as its children. Likewise, you can modify the same atribute for any given text line, using its type, class or ID.
